[Cramer's Rule](https://en.wikipedia.org/wiki/Cramer%27s_rule) 是一个解线性方程组的方法，它的原理是用行列式来解方程组。

原理是只要方程组的未知数个数和方程个数相等，且系数矩阵的行列式不为0，就可以用 Cramer's Rule 来解方程组。

```

```
